## 其它说明

* 该站点使用 `Cloudflare` 防护机制, 当出现 403 响应时, 可尝试将后端切换为 `curl_cffi` 并配置 `impersonate = "chrome136"`
* 切换后通常可正常访问, 但当 `request_interval < 3.0` 时, 仍可能再次触发 403
